# Algorithms Visualizer Program
![](Media/Front_Page.png)

This program's main idea is to visualize pathfinding algorithms on graphs.  
The three main algorithms for this project are:
* Dijkstra's algorithm.
* BFS algorithm.
* DFS algorithm.
